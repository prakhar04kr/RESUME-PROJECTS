🇮🇳 India Government Schemes Portal — Full Project Prompt

PROJECT OVERVIEW
Build a full-stack React web application called "YojanaConnect" (or "योजना कनेक्ट") — a smart, age-adaptive portal that helps Indian citizens discover central and state government schemes based on their age group, financial need, and life sector, with multi-language support.

PHASE 1 — AGE-ADAPTIVE UI SYSTEM
On first load, show an Age Detection / Selection Screen:

Ask user: "What is your age?" via a stylized input or slider
Optionally allow DOB entry
Detect and apply UI theme based on age group:


🎨 UI THEME SYSTEM (3 Modes)
Teen Mode (13–19 yrs)

Stack: React + TailwindCSS + Framer Motion
Aesthetic: Bold neon gradients (electric purple, hot pink, cyan), glassmorphism cards, animated particle background (use tsparticles)
Typography: Syne (headings) + DM Sans (body) from Google Fonts
Animations: Card flip reveals, bouncy hover effects, scrolling marquee with scheme names, glowing CTA buttons
Mascot: Animated cartoon character (SVG) guiding the teen
Vibe: Like a gaming UI meets Instagram Stories

Middle Age Mode (20–55 yrs)

Stack: React + Three.js (for 3D hero) + Framer Motion
Aesthetic: Deep navy + gold, professional 3D animated globe/building in the hero section using @react-three/fiber, smooth scroll transitions
Typography: Playfair Display (headings) + Source Sans 3 (body)
Animations: 3D card tilt on hover (react-tilt), animated counters for scheme stats, scroll-triggered section reveals
Vibe: Premium fintech/government dashboard feel

Senior Mode (56+ yrs)

Stack: React + plain CSS animations
Aesthetic: Soft whites, warm saffron/cream, large fonts (min 18px body, 28px headings), high contrast
Typography: Noto Serif Devanagari + Lora
Animations: Subtle fade-ins only, no jarring movement
Accessibility: aria-labels everywhere, keyboard-navigable, text-to-speech toggle using Web Speech API
Vibe: Newspaper clarity meets government trust

PHASE 5 — SCHEME DETAIL PAGE
Each scheme gets a full detail page (/scheme/:id) with:

Hero section: Scheme name, ministry logo, benefit highlight badge
Overview tab: What is the scheme, who benefits, why it exists
Eligibility tab: Checklist UI (✅ / ❌ based on user's profile)
Documents tab: Download checklist as PDF (react-to-print), document descriptions, where to get each one
How to Apply tab: Numbered step-by-step guide with screenshots/icons
Terms & Conditions tab: Accordion-style expandable T&Cs
Official Links: Styled button to official government portal
Share: WhatsApp / Copy link buttons (crucial for rural reach)
Related Schemes: 3 similar scheme cards

Scheme Card click → Detail page (tabs: Overview / Eligibility / Docs / Apply / T&Cs)
Eligibility Check: User fills mini-profile → Green/Red indicators on each scheme
Save Scheme: Bookmark to local profile (localStorage for now)
Share: WhatsApp deep link with scheme summary
